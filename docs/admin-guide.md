## 1. 관리자 페이지 접속하기

1. 웹 브라우저(크롬, 엣지 등)를 엽니다.
2. 주소창에 서비스 주소 뒤에 `/admin`을 붙여 입력합니다.
   - 예: `https://vod.letscareer.co.kr/admin`
3. 비밀번호 입력 화면이 나타납니다.
4. 관리자 비밀번호를 입력하고 **로그인** 버튼을 클릭합니다.
5. 로그인에 성공하면 **VOD 관리** 화면으로 이동합니다.

> **비밀번호를 모를 경우**: 개발 담당자에게 문의하세요. 비밀번호는 서버 환경변수(`ADMIN_PASSWORD`)에 저장되어 있습니다.

## 2. VOD 추가하기

VOD를 추가하면 참여자 화면에 즉시 표시됩니다.

### 유튜브 URL 복사하는 방법

1. 유튜브에서 올릴 영상을 엽니다.
2. 주소창의 URL 전체를 복사합니다.
   - 예: `https://www.youtube.com/watch?v=abcd1234`
   - 또는 공유 버튼 → **복사** 로 얻은 URL도 사용 가능합니다.

> **주의**: 유튜브 영상은 반드시 **일부 공개(링크가 있는 사용자만 볼 수 있음)** 또는 **공개** 상태여야 재생됩니다. **비공개** 영상은 서비스에서 재생되지 않습니다.

### VOD 추가 순서

1. 관리자 페이지 상단의 **VOD 추가** 섹션을 찾습니다.
2. 각 입력칸을 채웁니다.

   | 입력칸 | 설명 | 필수 여부 |
   |--------|------|-----------|
   | **제목** | 참여자에게 보이는 영상 제목 | 필수 |
   | **유튜브 URL** | 복사한 유튜브 주소 | 필수 |
   | **설명** | 영상 아래 표시되는 짧은 설명 | 선택 |
   | **제작일** | 영상 제작·촬영 날짜 | 선택 |

3. **추가** 버튼을 클릭합니다.
4. 아래 VOD 목록에 방금 추가한 영상이 나타나면 성공입니다.

## 3. VOD 공개·비공개 전환하기

삭제하지 않고 일시적으로 영상을 숨기고 싶을 때 사용합니다.

1. VOD 목록에서 원하는 영상을 찾습니다.
2. 해당 영상 오른쪽의 버튼을 확인합니다.
   - **숨기기** 버튼이 보이면 → 현재 **공개** 상태입니다.
   - **공개** 버튼이 보이면 → 현재 **비공개** 상태입니다.
3. 버튼을 클릭하면 상태가 즉시 전환됩니다.

비공개 처리된 VOD는 목록에서 회색으로 표시되고, **비공개** 태그가 붙습니다.
참여자 화면에서는 보이지 않습니다.

## 4. VOD 순서 변경하기

참여자 화면에 보이는 영상 순서를 조정할 수 있습니다.

1. VOD 목록에서 순서를 바꾸고 싶은 영상을 찾습니다.
2. 해당 영상 왼쪽의 **▲(위)** 또는 **▼(아래)** 버튼을 클릭합니다.
3. 클릭할 때마다 한 칸씩 이동합니다.
4. 목록 맨 위의 영상은 ▲ 버튼이 비활성화되고, 맨 아래 영상은 ▼ 버튼이 비활성화됩니다.

## 5. VOD 설명·제작일 수정하기

### 설명 수정

1. 수정하고 싶은 VOD 항목에서 설명 텍스트 부분을 클릭합니다.
   - 설명이 없으면 **설명 추가...** 라고 표시됩니다.
2. 텍스트 입력창이 나타납니다. 내용을 수정하거나 새로 입력합니다.
3. **저장** 버튼을 클릭합니다.
4. 취소하려면 **취소** 버튼을 클릭합니다.

### 제작일 수정

1. 수정하고 싶은 VOD 항목에서 제작일 부분을 클릭합니다.
   - 제작일이 없으면 **제작일 추가...** 라고 표시됩니다.
2. 날짜 선택창이 나타납니다. 원하는 날짜를 선택합니다.
3. **저장** 버튼을 클릭합니다.

## 6. VOD 삭제하기

> **주의**: 삭제하면 영상 정보가 완전히 제거됩니다. 단순히 숨기고 싶다면 [공개·비공개 전환](#3-vod-공개비공개-전환하기)을 사용하세요.

1. 삭제하려는 VOD 오른쪽의 **삭제** 버튼(빨간색)을 클릭합니다.
2. "삭제하시겠습니까?" 확인 창이 나타납니다.
3. **삭제** 버튼을 클릭하면 영구 삭제됩니다.
4. 취소하려면 **취소** 버튼을 클릭합니다.

## 7. 예외 유저 추가하기 (개별)

**예외 유저**란, 렛츠커리어 서버에 등록되지 않은 참여자지만 관리자가 직접 접근 권한을 부여한 유저입니다.

1. 관리자 페이지 하단의 **예외 접근 유저 관리** 섹션을 찾습니다.
2. **이름** 입력칸에 이름을 입력합니다.
3. **전화번호** 입력칸에 전화번호를 숫자만 입력합니다. (예: `01012345678`)
4. **추가** 버튼을 클릭합니다.
5. 아래 유저 목록에 추가된 유저가 나타납니다.

추가된 유저는 이름 + 전화번호를 로그인 화면에 입력하면 VOD에 접근할 수 있습니다.

## 8. 예외 유저 CSV로 일괄 등록하기

많은 인원을 한꺼번에 등록할 때 사용합니다.

### CSV 양식 다운로드

1. **양식 다운로드** 버튼을 클릭합니다.
2. `users_template.csv` 파일이 다운로드됩니다.

### CSV 파일 작성 방법

다운로드한 파일을 **엑셀** 또는 **메모장**으로 엽니다.

```
name,phoneNum
홍길동,01012345678
김철수,01098765432
이영희,01055556666
```

- 첫 번째 줄(헤더)은 수정하지 마세요: `name,phoneNum`
- 두 번째 줄부터 이름과 전화번호를 한 줄에 하나씩 입력합니다.
- 전화번호는 숫자만 입력합니다. (하이픈 `-` 없이)
- 파일을 저장할 때 **CSV(쉼표로 분리)** 형식으로 저장합니다.

### CSV 업로드

1. **CSV 업로드** 버튼을 클릭합니다.
2. 작성한 CSV 파일을 선택합니다.
3. 업로드가 완료되면 결과가 표시됩니다.
   - **추가**: 새로 등록된 인원 수
   - **스킵**: 이미 등록된 인원 수 (중복 제외)

## 9. 예외 유저 삭제하기

### 개별 삭제

1. 삭제하려는 유저 오른쪽의 **삭제** 버튼을 클릭합니다.
2. 확인 창에서 **삭제** 버튼을 클릭합니다.

### 여러 명 한꺼번에 삭제

1. 삭제하려는 유저 왼쪽의 체크박스를 클릭하여 선택합니다.
   - 상단 체크박스를 클릭하면 전체 선택/해제가 됩니다.
2. 파란 알림줄에 선택된 인원 수가 표시됩니다.
3. **선택 삭제** 버튼을 클릭합니다.

## 10. 배너 추가하기

배너는 VOD 목록 상단 또는 재생화면 하단에 표시되는 이미지입니다.
클릭 시 원하는 페이지로 이동하도록 설정할 수 있습니다.

### 배너 이미지 규격

| 항목 | 권장 사양 |
|------|-----------|
| 크기 | 1120 × 180 픽셀 |
| 형식 | JPG, PNG, WebP, GIF 등 일반 이미지 파일 |

### 배너 추가 순서

1. **배너 관리** 섹션을 찾습니다.
2. **클릭 시 이동할 URL** 입력칸에 배너를 클릭했을 때 이동할 주소를 입력합니다.
   - 예: `https://letscareer.co.kr/event`
   - 링크가 필요 없으면 비워두어도 됩니다.
3. **표시 위치**를 선택합니다.

   | 옵션 | 설명 |
   |------|------|
   | **목록 + 재생화면** | VOD 목록 상단과 재생화면 하단 모두에 표시 |
   | **목록 상단만** | VOD 목록 화면 상단에만 표시 |
   | **재생화면 하단만** | 영상 재생 화면 하단에만 표시 |

4. **랜덤 순서** 체크박스: 체크하면 이 배너가 포함된 슬라이드가 무작위 순서로 표시됩니다.
5. **이미지 선택 및 등록** 버튼을 클릭합니다.
6. 준비한 이미지 파일을 선택합니다.
7. 업로드가 완료되면 아래 배너 목록에 추가됩니다.

## 11. 배너 순서·삭제 관리하기

### 순서 변경

- 배너 목록에서 **▲(위)** / **▼(아래)** 버튼으로 순서를 조정합니다.
- 배너 슬라이드에 표시되는 순서가 변경됩니다.

### 배너 삭제

- 삭제하려는 배너 오른쪽의 **삭제** 버튼(빨간색)을 클릭합니다.
- 확인 없이 즉시 삭제되므로 주의하세요.

## 12. 로그아웃하기

1. 관리자 페이지 우측 상단의 **로그아웃** 버튼을 클릭합니다.
2. 관리자 로그인 화면으로 돌아갑니다.

> 브라우저를 닫아도 세션이 유지될 수 있으므로, 공용 컴퓨터에서 사용한 경우 반드시 로그아웃하세요.

## 13. 자주 묻는 질문 (FAQ)

### Q. VOD를 추가했는데 참여자 화면에 보이지 않아요.

**A.** 다음 항목을 확인하세요.
- VOD 항목에 **비공개** 태그가 붙어 있지 않은지 확인합니다. 붙어 있다면 **공개** 버튼을 클릭하세요.
- 유튜브 영상이 **비공개** 설정인 경우 재생되지 않습니다. 유튜브에서 영상 공개 설정을 **일부 공개** 또는 **공개**로 변경하세요.

---

### Q. 유튜브 URL을 입력했는데 "올바른 유튜브 URL이 아닙니다" 오류가 나요.

**A.** 아래 형식의 URL을 사용하세요.
- `https://www.youtube.com/watch?v=영상ID`
- `https://youtu.be/영상ID`

유튜브 쇼츠(`/shorts/`) URL은 지원되지 않을 수 있습니다.

---

### Q. 예외 유저를 추가했는데 로그인이 안 된다고 해요.

**A.** 다음을 확인하세요.
- 이름이 정확히 일치하는지 확인합니다. (공백 포함)
- 전화번호가 숫자만으로 저장되어 있는지 확인합니다. (하이픈 없이 `01012345678` 형식)
- 관리자 페이지에서 해당 유저가 목록에 있는지 다시 확인합니다.

---

### Q. CSV 업로드 후 일부 인원이 스킵되었어요.

**A.** 스킵은 이미 등록된 유저(이름 + 전화번호가 동일)를 중복 등록하지 않기 위한 정상 동작입니다.
오류 목록에 표시된 항목만 개별적으로 확인하세요.

---

### Q. 배너 이미지가 이상하게 표시돼요.

**A.** 권장 이미지 크기는 **1120 × 180 픽셀**입니다.
크기가 다를 경우 이미지가 늘어나거나 잘릴 수 있습니다. 이미지 편집 프로그램으로 크기를 맞춰서 다시 업로드하세요.

---

### Q. 관리자 비밀번호를 바꾸고 싶어요.

**A.** 비밀번호는 서버 환경변수에서 관리됩니다. 개발 담당자에게 환경변수 `ADMIN_PASSWORD` 값 변경을 요청하세요. Vercel 대시보드에서 변경 후 재배포가 필요합니다.
